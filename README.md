## Requirements

- [mcamara/laravel-localization](https://github.com/mcamara/laravel-localization)

## Installation

- Require it with Composer:
```bash
composer require lwwcas/help
```

- Run the command
```bash
composer dump-autoload
```

## Simple usage

```php
echo currentPage();
// Return to current page
```
